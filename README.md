# FedCache

This repository is the official Pytorch implementation DEMO of **FedCache**:

**FedCache: A Knowledge Cache-driven Federated Learning Architecture for Personalized Edge Intelligence**

--------------------------
## Requirements:
- Python:  3.10
- Pytorch:  1.13.1
- torchvision:  0.14.1
- hnswlib
- Other dependencies

-------
## Run this DEMO
```python main_fedcache.py```
